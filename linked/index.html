背包、队列、栈的链表实现

2016-07-14 17:23:04

链表

链表是一种应用广泛的抽象数据结构, Java使用引用将分散的节点”链接”在一起,
构成一个整体–链表. 链表可以作为数组的替代, 而且相比于数组,
向链表中插入和删除数据都更加方便. 实际情况下, 链式存储结构的应用会更广泛,
毕竟可以更灵活的运用内存.

链表抽象数据类型的定义

使用范型和嵌套类定义链表结构是非常简洁的:

private class Node
{
    Item item;  // 存储
    Node next;  // 引用
}

注意⚠

  1. 节点类Node定义为private, 因为节点是作为内部对象使用的
  2. Node实例化的对象, 是指向特定节点的引用

一些对链表的操作

表头操作

添加表头

在链表中添加表头是十分方便的

Node oldfirst = first;  // 新建oldfirst引用, 保存旧的first node
Node first = new Node();  // 新的first节点
first.item = item;  // 数据存储
first.next = oldfirst;  // 指向旧的first节点

添加表头数据和链表长度是没有关系的, 时间复杂度是常数时间.

删除表头

在链表中删除表头是十分方便的

first = first.next;  // 把first指向的第二个节点设为第一个节点

删除表头是常数时间.

表尾操作

添加表尾

在链表中添加表尾是十分方便的,
我们只需要将最后一个last节点的next指向新的last节点(原来是null)

Node oldlast = last;
Node last = new Node();
last.item = item;
oldlast.next = last;

添加表尾是常数时间.

删除表尾

删除表尾就比较麻烦了, 因为删除表尾需要知道倒数第二个元素,
也就是需要对列表进行遍历, 遍历的时间复杂度是与链表的长度成正比的, 很耗时.
所以一般避免对链表的表尾进行删除操作.

使用双向链表可以避免这个问题;)

实现

栈

public class Stack<Item>
{
    private Node first;  // 栈顶
    private int N;  // 栈种元素的数量
    private class Node
    {
        Item item;
        Node next;
    }
    public boolean isEmpty()
        { return first == null; }
    public int size()
        { return N; }
    public void push(Item item)
    {
        // 向表头插入数据
        Node oldfirst = first;
        first = new Node();
        first.item = item;
        first.next = oldfirst;
        N++;
    }
    public Item pop()
    {
        // 删除表头数据
        Item item = first.item;
        first.next = first;
        N--;
        return item;
    }
}

通过对表头的操作, 使得push和pop都是常数时间, 而且, 链表的大小是动态的,
比起上一篇博客中费劲心思的数组扩容, 链表简直是完美.

队列

public class Queue<Item>
{
    private Node last;  // 出队列
    private Node first; // 入队列
    private int N;  // 队列计数
    private class Node
    {
        Item item;
        Node next;
    }
    public boolean isEmpty()
        { return N == 0; }
    public int size()
        { return N; }
    public void enqueue(Item item)
    {
        // 向表尾添加数据
        Node oldlast = last;
        last = new Node();
        last.item = item;
        oldlast.next = last;
        N++;
    }
    public Item dequeue()
    {
        // 删除表头数据
        Item item = first.item;
        first = first.next;
        N--;
        return item;
    }
}

import java.util.iterator
public class Bag<Item> implements Iterable<Item>
{
    private Node first;
    private class Node
    {
        Item item;
        Node next;
    }
    public void add(Item item)
    {
        Node oldfirst = first;
        first = new Node();
        first.item = item;
        first.next = oldfirst;
    }
    public Iterator<Item> iterator()
    {
        // 实现Iterable接口
        // 返回一个Iterator对象
        return new ListIterator();
    }
    public class ListIterator implements Iterator<Item>
    {
        // ListIterator迭代器
        // 实现Iterator迭代器接口: hasNext, next, remove
        private Node current = first;
        public boolean hasNext()
            { return current != null; }
        public void remove() {}
        public Item next()
        {
            // 迭代器遍历方法
            Item item = current.item;
            current = current.next;
            return item;
        }
    }
}
