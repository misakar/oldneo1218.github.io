使用pudb调试Python程序

2016-06-19 09:36:13

今天尝试用pudb调试Python程序, 当然我用的很肤浅, 还是在这里做个纪录。

1. PUDB

GDB ==> PDB ==> PUDB

2. 安装PUDB

pip install pudb

3. 使用PUDB

Python作为动态语言, 调试起来是比较困难的, 简单的调试就是在程序执行过程中不断的print, print, print。而pudb提供了图形化界面, 可以设置断点、侧边栏显示调用栈和变量, 用起来比较方便。其实第一次见到pudb界面时感觉回到了上个世纪haha。

1. 使用pudb调试

在你需要调试的程序中, 加入下面2行代码

from pudb import set_trace; set_trace()

然后

$ pudb bp.py

pudb需要打开某些文件, 所以如果出现permission错误需要 sudo pudb bp.py

2. 基本调试

基本调试很简单(我用的比较肤浅...), n就是next, 旁边的Variables实时显示变量变化, Stack显示调用栈, 遇到断点停止。如果遇到终端输出则跳转到终端。s则是进入函数内部, 这是Stack和Variables就会显示函数内部的变量和调用栈的变化。我录了一下屏:
